DOM Level 1

DOM Core - в данном модуле описывается способ представления XML-документа и реализован доступ к любой его части.
DOM HTML - здесь реализованы механизмы характерные для работы HTML - документа.

DOM Level 2

DOM Core - расширено.

Добавлены модули:

DOM Views - интерфейсы для отслеживания различных представлений документа(например до и после применения стилей CSS).
DOM Events - интерфейсы для событий и обработки событий.
DOM Style - Интерфейсы для работы со стилизацией элементов с помощью CSS.
DOM Traversal and Range - интерфейсы для обхода элементов дерева документа и выполнения операций над ними.

BOM

- функция отображения всплывающих окон в браузере.
- возможность перемещать, закрывать и изменять размеры окна браузера.
- объект navigator предоставляет подробные сведения о браузере.
- объект location, предоставляет подробные сведения о странице загруженной в браузере.
- объект screen, предоставляет подробные сведения о разрешении экрана.
- поддержка coockie-файлов.
- настраиваемые объекты, включая XMLHttpRequest, а также ActiveXObject в Internet Explorer.
